# Valentines

Small interactive Valentine web page built with HTML, CSS, and JavaScript.

## Run Locally

Open [`index.html`](./index.html) in a browser. No build step or package installation is required.

## Files

- `index.html` and `styles.css` render the first page.
- `script.js` handles the interaction.
- `yes_page.html` and `yes_style.css` render the follow-up page.
- `assets/heart_bg.mp4` provides the background video.

## Project Notes

This is a static browser project. Keep media assets compressed so the page remains quick to load when hosted from GitHub Pages or another static host.
